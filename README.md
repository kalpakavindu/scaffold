# Scaffold

Starter templates for all my projects.

## Overview

* `/react-vite-ts` - React+Vite template with TypeScript for frontend web projects.
* `/express-mongoose-js` - Express+Mongoose template with JavaScript for server side API projects.
* `/cmake-cpp` - CMake template with builtin presets for VisualStudio and MinGW toolchains for C++ and C projects.
